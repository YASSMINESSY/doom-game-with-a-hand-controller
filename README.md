# 🎮 DOOM Terminal Game – Python OOP Edition

A DOOM-inspired terminal game built in **Python**, controlled using a **game controller** instead of a keyboard.  
This project focuses on **Object-Oriented Programming (OOP)**, clean architecture, and immersive terminal gameplay.

---

## 🚀 Features
- 🎮 **Controller-based gameplay** (no keyboard needed)
- 🖥️ Runs directly in the **terminal**
- 🧠 Built with **Python & OOP principles**
- 🧱 Modular and extensible game architecture
- 🔥 Multiple gameplay mechanics and custom additions

---

## 🕹️ Controls
- Game is controlled using a **hand controller**
- Each button is mapped to a specific in-game action
- Keyboard input is intentionally minimized to enhance immersion

---

## 🛠️ Technologies Used
- **Python**
- **Object-Oriented Programming (OOP)**
- Terminal-based rendering
- Controller input handling
